Analyse-Profile

Innerhalb der Profile-View werden alle gespeicherten Analyse-Profile angezeigt. Innerhalb eines Analyse-Profils können benutzerdefinierte Charts angelegt werden. Profile werden innerhalb des Workspaces gespeichert und können exportiert, importiert werden.

Erstellen eines Analyse-Profils

  1. New-Wizard öffnen
  2. Kategorie Garbage Collection Analysis öffnen
  3. Analysis Profil auswählen
  4. JVM-Vendor und Name angeben
  5. Fertigstellen
