Analyse

Die Analyse eines Garbage Collection Logs kann zwar mit dem Standard Profil gemacht werden, es wird aber die Verwendung von benutzerdefinierten Profilen empfohlen. Benutzerdefinierte Profile ermöglichen das Hinzufügen und Definieren von eigenen Charts.

Für weitere Informationen bitte konsultieren Sie das standard Profil und benutzerdefinierte Profile.
